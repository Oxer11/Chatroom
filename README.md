## How to use

### Prerequisites

```
Python 3.7
PyQt 5.0
```

### Usage

```bash
# Clone this repository
$ git clone https://github.com/Oxer11/Family-of-Calabash-Brothers.git

# Go into the repository
$ cd Family-of-Calabash-Brothers

# Run server
$ python server.py

# Run client
$ python client.py

```
